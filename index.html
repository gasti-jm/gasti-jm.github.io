Argentum Online en Java

Migración de videojuego MMORPG 2D de VB6 a Java, utiliza la librería grafica OpenGL en Java conocida como LWJGL3. Permite la conexión a su servidor original de VB6 utilizando sockets de la API de Java y utiliza ImGui para el diseño de interfaz de usuario. Además de ser una migración exitosa este mismo mejora el rendimiento y la estructura de su código a comparación con el original.

Ver codigo en GitHub

<sobre-mi>

Soy programador desde los 12 años, cuando comencé desarrollando videojuegos. Mi primer lenguaje fue Visual Basic 6.0, lo que me permitió adentrarme en el mundo de la programación y aprender los conceptos básicos a través de un foro activo llamado GS-Zone. Después de varios años, decidí formalizar mis conocimientos al iniciar una carrera universitaria, lo que me permitió profundizar en diversas áreas del desarrollo de software. Tras finalizar la carrera, continué aprendiendo de forma autodidacta, eligiendo un stack tecnológico que me permitiera implementar y explorar nuevas tecnologías. Actualmente, me desempeño como desarrollador Full-stack especializado en Java.
