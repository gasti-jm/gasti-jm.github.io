Argentum Online en Java

Migración de videojuego MMORPG 2D de VB6 a Java, utiliza la librería grafica OpenGL en Java conocida como LWJGL3. Permite la conexión a su servidor original de VB6 utilizando sockets de la API de Java y utiliza ImGui para el diseño de interfaz de usuario. Además de ser una migración exitosa este mismo mejora el rendimiento y la estructura de su código a comparación con el original. Cumple con la licencia GNU GPL3 y es de open source.
